# textual-dataviz

[Claude Code](https://claude.com/claude-code) 用のASCII/Unicodeテキストベースデータ可視化スキル。

会話コンテキスト内のデータ種別を自動判定し、ブロック文字と絵文字を使ったターミナル表示可能な可視化を生成します。

## チャートタイプ

| タイプ | 用途 | 出力例 |
|-------|------|--------|
| 横棒グラフ | 比率・割合 | `食費 █████░░░ 65%` |
| トレンドチャート | 時系列推移 | `Week 1: ████░░ 40% ↗️` |
| ヒートマップ | 完了/未完了 | `✅✅⬜✅✅⬜✅` |
| ツリー | ディレクトリ・階層 | `├── src/` |
| 比較テーブル | 前月比・前年比 | Markdown表 + 差分絵文字 |
| 集計チャート | 分布・ランキング | `████ 45件` |
| ステータス | 段階的進捗 | `🟥🟧🟨🟩 (0-100%)` |
| フロー図 | ワークフロー・依存関係 | `A → B → C` |

## インストール

### Claude Code プラグイン経由（推奨）

```bash
claude plugin install textual-dataviz
```

### 手動インストール

```bash
git clone https://github.com/3062-in-zamud/textual-dataviz.git
cd textual-dataviz
bash install.sh
```

## 使い方

### トリガーワード

以下の言葉でスキルが自動発動します:
- 日本語: 「可視化して」「グラフにして」「図にして」「チャートで見せて」
- 英語: "visualize", "show as chart", "draw diagram"

### スラッシュコマンド

```
/textual-dataviz type=bar
/textual-dataviz "ディレクトリ構造を表示"
```

### パラメータ

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `type` | 自動判定 | チャートタイプ（bar/trend/heatmap/tree/table/status/flow） |
| `output` | ターミナル | 出力先（terminal/file/daily-note） |
| `width` | 20 | チャート幅（文字数） |
| `target` | コンテキスト | 可視化対象データを明示指定 |
